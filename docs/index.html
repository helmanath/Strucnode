Strucnode

Visual file organizer · Python · Tkinter

Python 3.10+ Tkinter Node Editor MIT License

Overview

A complete file organization app
with a visual node editor

🗺️

Explorer

Browse folders, preview media, inspect metadata, and explore files with integrated viewers

🔗

Node Editor

Build destination folder structures visually by chaining metadata nodes, connectors, and folder levels

📁

Organize

Preview and apply copy or move operations with extension filters and duplicate-handling strategies

⚡ Getting started

Install in 3 steps

1 Clone the repository
git clone https://github.com/helmanath/Strucnode.git
cd Strucnode
2 Create a virtual environment
python -m venv .venv
source .venv/bin/activate # Linux/macOS
.venv\Scripts\activate # Windows
3 Install and run
pip install -r requirements.txt
python strucnode.py

Optional dependencies

Pillow Thumbnails & RAW
opencv-python Video playback
rawpy + exifread RAW / EXIF files
sounddevice + moviepy Video audio

✓ No mandatory extra dependencies

The core app (Explorer + Node Editor) runs with only Python 3.10+ and Tkinter, which is included in the standard library.

Tab 1

File Explorer

🖼️ Advanced preview

Images, videos, and audio with integrated controls — JPEG, PNG, MP4, RAW/CR2/NEF, 360°…

📊 EXIF metadata

Read full EXIF data: camera, ISO, focal length, aperture, capture date

🌐 360° viewer

Automatic equirectangular image detection with interactive panoramic navigation

📹 Full video playback

Playback with audio, volume control, progress bar, and fullscreen mode

🎨 Color coding by file type

Images, videos, audio, code, docs, archives — each category has its own color

✂️ Image tools

Rotation, horizontal/vertical flip, zoom and pan, fullscreen, RAW decoding via rawpy or dcraw

Tab 2 — ✨ Key feature

Visual Node Editor

Build the target folder structure visually by chaining nodes on an interactive canvas

🟤

Folder nodes

Each node = one hierarchical level in the destination tree

🔗

Connector nodes

Fixed text between values: dash, underscore, space, dot, or custom

🏷️

Argument nodes

EXIF date, creation year/month/day, filename, extension, category, size…

🔄

Connections

Click ports to connect nodes — click a wire to cut it

💾

Presets

Save and load JSON structures — automatic reload on startup

👁️

Preview live

Real-time preview of the generated structure in a side panel

Example

Sort photos in your folder by date

Node structure:

📅 EXIF Year → Liant " - " → 📅 EXIF Month
↓ connected to the "Name" port of:
📁 Folder

Generated structure:

📁 destination/
  📁 2024 - 01/
    IMG_0042.JPG
    IMG_0043.JPG
  📁 2024 - 03/
    DSC_1823.NEF
  📁 2025 - 07/
    photo_vacances.jpg

💡 Other available arguments: Filename, Extension, Category (Images/Videos/…), Size, First letter, Modified/Created date

Tab 3

Organize — Copy / Move

🚀 Apply structure

Copy or move indexed files to a destination by applying the defined node-based structure

⚡ Multithreaded

Background execution with a progress bar — the interface stays responsive

⚠️ Duplicate management

If files already exist at the destination, a dialog offers 3 actions:

IGNORE Skip conflicting files and copy the remaining ones
REPLACE Overwrite existing files at the destination
RENAME Add -doublon to the destination filename
CANCEL Cancel the entire operation without changing anything if you made a mistake
