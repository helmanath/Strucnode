# Code of Conduct

## Our Pledge

We aim to keep this project respectful, helpful, and welcoming.

## Expected Behavior

- Be constructive.
- Be respectful.
- Keep feedback focused on the code and the project.
- Assume good intent.

## Unacceptable Behavior

- Harassment.
- Personal attacks.
- Offensive language.
- Trolling or disruptive behavior.
